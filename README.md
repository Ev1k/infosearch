### 1 задание
Сделан парсер для нахождения ссылок с сайта https://dic.academic.ru + выкачка html-файлов

### 2 задание
Из сохраненных документов 1.txt-200.txt были ивлечен весь текст, были выделены отдельные слова и произведена лемматизация. Были созданы отдельные папки tokens/lemmas. Также созданы файлы lemmas_index.txt и tokens_index.txt со всеми токенами и леммами

### 3 задание
Из файлов директории lemmas были взяты леммы и построен инвертированный список inverted_index.txt (task_3_build_index.py).
По инверсированному списку производится поиск запроса пользователя. В ответ программа выдаёт список соответствующих документов (task_3_bool_search.py).

### 4 задание
Из документов из папки text был извлечен текст и токенизирован. По спискам терминов и лемм из файлов tokens_index.txt и lemmas_index.txt был посчитан tf и idf для терминов и лемм. Результаты сохранены в lemmas_tfidf и terms_tfidf по каждому документу

### 5 задание
Пользовательский запрос токенизируется и лемматизируется. После этого для него строится вектор в пространстве лемм с весами tf-idf, где idf берётся из уже построенного индекса документов. Далее для каждого документа вычисляется косинусное сходство между вектором запроса и вектором документа. Результаты сортируются по убыванию значения сходства, и пользователю выводятся top-k наиболее релевантных документов.

### 6 задание
Добавлен WEB-интерфейс для векторного поиска и ранжирования (`task_6_web.py`).
По введённому запросу система выводит top-10 документов, отсортированных по косинусному сходству.

Запуск:
`python3 task_6_web.py`

После запуска открыть в браузере:
`http://127.0.0.1:8000`
